# SejioMedia
A Minecraft Fabric mod that renders YouTube videos onto a placeable in-game block, with audio playback included.

![Minecraft 1.21](https://img.shields.io/badge/Minecraft-1.21-brightgreen)

## NOTICE
Source code may not always reflect the latest release build.

## Requirements
- Minecraft 1.21
- [Fabric Loader](https://fabricmc.net/use/installer/)
- [Fabric API](https://modrinth.com/mod/fabric-api)
- Windows only — Mac and Linux are not currently supported

---

## Installation
1. Install Fabric and Fabric API
2. Place the SejioMedia `.jar` into your `mods/` folder
3. Launch the game — yt-dlp and FFmpeg will download automatically on first use
4. Place a Media Screen block, right-click it, paste a YouTube URL, and press Play

---

## Current Limitations
- Singleplayer only — multiplayer support is planned
- Early release, expect bugs and rough edges
